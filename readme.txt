== Description ==

Automatically the page or post slug as a class of the body prefixed with "slug-"

== Installation ==

Upload the plugin folder to the wp-content/plugins directory

Enable "Tomi Post Slug As Class" in the plugins section

== Instructions ==

No need for any further configuration.
You will now have the slug of the post/page being rendered added as an extra class to the body tag.
(The class will be prefixed with "slug-" so you can easily identify the class and also so you can later check to see what styles you have that use this plugin in order to get applied)
